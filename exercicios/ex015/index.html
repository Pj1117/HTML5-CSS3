**A tag <style> funciona com seletores (serve para body, h1, h2, p etc) e a linha dentro das chaves se chama declaração.
ex:
<style>
    body {
        background-color: blue; font-family: Arial...; font-size: 17px
    }
</style>

**Estilo Inline tem a limitação de só ser aplicado dentro da tag <style>

**O parâmetro font-family muda o estilo de fonte de todo o texto do site

**O parâmetro text-align, se colocado dentro do body, muda a formatação do texto do site INTEIRO!(justificado, centralizado...) se colocado dentro de um parágrafo, só é alterada a formatação desse mesmo parágrafo.

**Toda tag escrita dentro de <style> é considerada CSS por ser uma tag de Estilo
