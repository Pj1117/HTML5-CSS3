Media Types


**media="screen" - Somente para a tela
**media="all" - Tudo.
**media="print" - Para impressora


Media Features


**media types são escritas como acima, media features são escritas entre parênteses.
EX: media="screen and (orientation:portrait)


Media Query


**A media query é a mesma coisa que media type (tipo de midia) + media features (característica de midia, pois exeistem tamanhos de telas diferentes).
