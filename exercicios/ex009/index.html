Listas Ordenadas

**São aquelas que a ordem faz diferença

**A tag <ol> (Ordened List/Lista ordenada), ordena os itens dentro de uma LISTA criada com a tag <li>.

**O parâmetro 'type' dá varias opções de ordenamento dos itens da lista se ela vai ser ordenada por números, letras, números romanos etc e o parâmetro 'start' permite colocar qual elemento dentro do que foi escolhido antes serà o primeiro ex: start '2' (vai iniciar pelo número 2 daquele conjunto de elementos).

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escovar os dentes

Listas não ordenadas

**Significado: ul(unordened list)

**Marca os itens dentro da lista(a ordem NÃO importa)

**Tambêm dá para alterar o tip, porém, só existem três tipos: 'square', 'circle' e 'disk'.
