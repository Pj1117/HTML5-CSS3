**Utilizando a tag <optgroup> é possível criar um grupo de opçoes como regiões, estados etc, utilizando o parâmetro label na linha do <optgroup>.

**Para fazer que uma opção já venha selecionada automaticamente, é só utilizar o parâmtro selected dentro da <option> que será selecionada (por padrão a primeira vem selecionada).

**Para colocar elementos que não estão listados nas opções, é necessário usar a tag <datalist> com um id e esse id será colocado no parâmetro list dentro do input. As <option>(s) dentro dessa tag e o value dentro da tag <option> é opcional neste caso.

**<textarea> é infinito, não existe número de linhas máximo

**É recomendado usar o método post se for usar textarea para não ter problema de limitação.
