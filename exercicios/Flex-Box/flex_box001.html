Flex box

**display: flex; Adiciona um container como flex

**flex-direction: Define a direção dos itens, com o padrão sendo row (linha).

**flex-diretion: row-reverse: Define os itens em linha só que da direita para a esquerda. (o mesmo se aplica para a coluna column-reverse).

**flex-wrap: nowrap; Vai definir se os itens vão ter uma quebra de linha ou não, tendo o valor padrão nowrap

**flex-wrap: reverse: Permite a quebra de linhas, da esquerda para a direita.

**flex-flow: É uma shorthand que permite escrever o **flex-direction e o flex-wrap.

**justify-content: center; Serve para fazer um alinhamento Horizontal, com o valor padrão sendo flex-start (direita e esquerda)

**align-itens: Vai fazer um alinhamento Vertical dos itens, com o valor padrão sendo stretch que define que os itens cresçam igualmente. (cima e embaixo)

**align-itens: baseline; Vai fazer o alinhamento Vertical de acordo com o conteúdo de cada item.

**align-content: ; Vai fazer um alinhamento Vertical, mas só quando os itens estiverem em uma multi-line (quebra)
